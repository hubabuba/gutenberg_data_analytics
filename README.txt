****************************
* Software Packages        *
****************************

The following packages are required to run the software

Software was written on python 2.7.15, Anaconda distribution

- Natural Language Tool Kit (NLTK)
- Gensim
- Scitkit-Learn
- Numpy
- Pandas
- Matplotlib


****************************
* Data	                   *
****************************

All data is in the folder data, available on Google Drive
https://drive.google.com/drive/folders/1SHSG8dwdpQwrYXOJXQpAq7iLWlPVhybT?usp=sharing

Folder should be in the same directory as the iPython Notebook


****************************
* Software	           *
****************************

Some of the software was put in separate files to provide better encapulsation. The files are imported in the iPython notebook.
Documentation for the software is provided in the notebook. For installation, the files have to be in a Folder ProjectLibrary, which is located in the same folder as the notebok.
